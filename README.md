# Mobile Sales Dashboard - 4 Year Sales Analysis

## Overview

The Mobile Sales Dashboard is a data analytics project developed using Power BI to analyze and visualize mobile sales data collected over four years. The dashboard provides valuable insights into sales performance, customer behavior, product popularity, and regional trends, helping businesses make informed decisions based on data.

## Project Objectives

* Analyze mobile sales performance over a four-year period.
* Identify top-performing mobile brands and models.
* Understand customer purchasing patterns.
* Monitor sales trends across different regions and time periods.
* Create an interactive dashboard for easy business reporting and decision-making.

## Dataset Information

The dataset contains mobile sales transaction records covering four years. The data includes:

* Order ID
* Sale Date
* Mobile Brand
* Mobile Model
* Quantity Sold
* Unit Price
* Total Revenue
* Customer Information
* Sales Region
* Payment Method

## Data Collection

The sales data was collected from transaction records and consolidated into a single dataset for analysis. Historical sales information from multiple years was included to enable trend analysis and performance comparison.

## Data Cleaning and Preprocessing

The following data preparation steps were performed:

* Removed duplicate records.
* Handled missing and null values.
* Corrected inconsistent brand and model names.
* Standardized date formats.
* Verified sales and revenue calculations.
* Converted columns to appropriate data types.
* Created calculated columns and measures for analysis.

These steps ensured data accuracy and reliability before visualization.

## Key Insights

* Sales increased steadily over the four-year period.
* A few leading brands generated the majority of revenue.
* Seasonal sales peaks were observed during specific months.
* Certain regions consistently outperformed others in sales volume and revenue.
* Premium mobile models generated higher revenue despite lower sales quantities.
* Customer purchasing patterns revealed strong brand preferences.
